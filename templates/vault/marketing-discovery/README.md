# Daily Marketing Discovery — твой vault

Этот vault настроен для работы с `/daily-discovery`.

## Структура

```
marketing-discovery/
├── psyche/QUEUE.md     # 73 принципа из скилла (трекер выполнения)
├── psyche/POOL.md      # 24 внешних принципа (Halo Effect, Pre-suasion...)
├── psyche/YYYY-MM-DD-{principle}.md  # Заметка дня
├── extraction/         # Product Hunt + GitHub Trending
├── radar/              # Рекламные кейсы
└── ideas/              # Кросс-синтез
```

## Установка

```env
# .env в корне проекта
TOLARIA_VAULT_PATH=путь/к/этому/vault
```
